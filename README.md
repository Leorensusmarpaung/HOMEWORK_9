HOMEWORK_9 - SauceDemo Automation Testing

 Project Overview
This project is an automation testing assignment for SauceDemo website using Selenium WebDriver with JavaScript and Mocha framework.

Test Cases
1. Successful Login
- Open website https://www.saucedemo.com/
- Input valid username: `standard_user`
- Input valid password: `secret_sauce`
- Click login button
- Verify user successfully enters inventory page

2. Sort Products from A-Z
- Access product inventory page after login
- Select sorting dropdown
- Choose `Name (A to Z)`
- Verify products are sorted alphabetically

Tech Stack
- JavaScript
- Node.js
- Selenium WebDriver
- Mocha
- ChromeDriver
